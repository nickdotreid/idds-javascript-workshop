Programming is important because....

There isn't a 'great' reason why programming is important, but programs and software have become pervasive in our lives (I'm sure you can think of a few examples). There are many different ways to use programs (or software), but a common thread is to simplify tasks.

What is Javascript?

Javascript is a programming language that was initally made to add simple interactivity to webpage — since then this language has been widely adopted.

There are many different programming languages — many people are firecly defencive of their 'favorite' language, but there isn't a clear 'better' or 'worse' language more so than French is better than Tamli. Some languages are better for certain tasks, and others are better because of the extra libraries that are available.

We chose Javascript for this workshop because it's a forgiving language with a simple syntax and is built to run in a web browser (meaning you don't need to do any Yak-Shaving to get started). Many of the fundamentals from this workshop will be transferable to other programming languages, so keep in mind this is just the begining.

Drawing with Logo
